¿Qué es una variable? 📦

Una variable es un espacio en la memoria del computador que permite almacenar información para ser utilizada y modificada posteriormente. Puedes imaginarla como una caja etiquetada donde guardas datos esenciales para que tu programa funcione. ¡Es fundamental para crear programas dinámicos!




# Asignación de valores a variables
nombre = "Juan"
edad = 18

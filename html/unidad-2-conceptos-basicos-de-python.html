Ecosistema y Sintaxis del Lenguaje Python

¿Qué es Python?

Python es un lenguaje de programación de propósito general, alto nivel e interpretado. Su filosofía se centra en la legibilidad del código y la productividad del desarrollador. Es la herramienta líder en la Industria 4.0, siendo el estándar de facto para Inteligencia Artificial, Ciencia de Datos y Automatización de Infraestructura.

Atributos Técnicos del Lenguaje

✅ Sintaxis Limpia: Código cercano al lenguaje humano.
🌍 Multiplataforma: Ejecución en Windows, Linux y macOS.
📦 Ecosistema: Miles de librerías listas para usar.

Primer Script: Estructura de Salida

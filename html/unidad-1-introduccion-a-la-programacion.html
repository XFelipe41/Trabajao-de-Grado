¿Qué es la programación?

Es el proceso estratégico de diseñar y codificar instrucciones estructuradas para que una computadora ejecute tareas específicas. Es el puente de comunicación entre la lógica humana y la capacidad de procesamiento de la máquina.

Glosario Técnico de la Unidad

  • Algoritmo: Secuencia lógica de pasos para resolver un problema de forma finita.
  • Lenguaje de Programación: Sintaxis formal utilizada para dar órdenes a la CPU.
  • Código Fuente: El manuscrito original del programa escrito por el desarrollador.
  • Compilación: Proceso de traducción del código a lenguaje binario (máquina).

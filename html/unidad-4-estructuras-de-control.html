Estructuras de Control: Flujo y Lógica

Condicionales 🔀

Las estructuras condicionales (como if, elif y else) permiten a tu programa tomar decisiones. Evalúan una condición (que resulta en verdadero o falso) y ejecutan un bloque de código u otro dependiendo del resultado.





edad = 18

if edad >= 18:
    print("Eres mayor de edad")

Ciclos y Bucles 🔄

Los ciclos (como for y while) permiten repetir una serie de acciones o bloques de código múltiples veces. Son ideales para automatizar tareas repetitivas o recorrer listas de datos.





# Este ciclo imprimirá los números del 0 al 4
for i in range(5):
    print(i)

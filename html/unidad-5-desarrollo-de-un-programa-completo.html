UNIDAD 5

Desarrollo de un Programa Completo

Integrando el conocimiento 🧩

Es el momento de unir las variables, los tipos de datos y el flujo de control. Un programa completo interactúa con el usuario, procesa la información y devuelve un resultado útil.

📥 Entrada ➡️ ⚙️ Proceso ➡️ 📤 Salida

Ejemplo: Calculadora Básica




# Solicitud de datos (Entrada)
a = int(input("Número 1: "))
b = int(input("Número 2: "))

# Lógica (Proceso)
suma = a + b

# Impresión (Salida)
print("Resultado:", suma)

🚨 Zona de Retos: Modo Debug

Un excelente programador es, ante todo, un analista de código roto. A continuación, tienes **3 retos lógicos** con bloques de código de Python que contienen errores habituales de compilación. Identifica el error, haz tu diagnóstico y comprueba tu solución.

🚨 RETO 1: Bucle roto e indentación errónea

El siguiente script debería iterar del 1 al 5 e imprimir si cada número es par o impar, pero presenta múltiples fallos:

for i in range(1, 6) if i % 2 = 0: print("El número " + i + " es par") else: print("El número " + i + " es impar")

¿Qué errores graves impiden la ejecución de este código? (Selecciona todas las que apliquen):

  • ✓
    Falta el carácter dos puntos (:) al declarar el ciclo for.
  • ✓
    El condicional utiliza el operador de asignación simple = en lugar del operador de comparación relacional ==.
  • ✓
    La línea del primer print tiene un error de indentación (no está dentro del bloque del if).
  • ✓
    Intentar concatenar una cadena con el entero i causará un TypeError (debe usarse str(i) o un f-string).

🚨 RETO 2: Error de Tipos (TypeError)

El siguiente programa le solicita la edad al usuario para verificar si cumple la mayoría de edad laboral, pero arroja un error al ingresar un número:

edad = input("Ingresa tu edad: ") if edad >= 18: print("Acceso concedido al sistema técnico") else: print("Acceso denegado: requieres ser mayor de edad")

¿Por qué fallará este programa de forma catastrófica al evaluar la edad?

  • ✓
    La función input() retorna texto (string). Comparar un string con el entero 18 da un TypeError. Falta castear la entrada con int().
  • ✓
    La variable edad no está declarada de forma explícita con su tipo de datos al principio del programa.
  • ✓
    Las comillas del comando print están mal cerradas y generan un SyntaxError en la consola.

🚨 RETO 3: Ámbito de Variables (NameError)

Este código define una función para calcular el área de un componente tecnológico, pero el intérprete se detiene al intentar imprimir el resultado:

def calcular_area(base, altura): area = base * altura return area print("El área resultante es:", area)

¿Qué error informará la terminal de Python y por qué?

  • ✓
    TypeError en el print porque no se pueden concatenar textos con áreas matemáticas.
  • ✓
    NameError porque area es una variable de ámbito local dentro de la función y no existe en el contexto global desde el que es invocada.
  • ✓
    SyntaxError porque la palabra reservada return no puede retornar variables directamente.
